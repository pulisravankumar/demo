Overview

Stone, Paper, Scissors is a beginner-friendly Python project that helps you understand how to use logic, randomization, and user interaction. In this project, the computer becomes one player, and you (the user) are the other player.

Stone (Rock), Paper, Scissors is a classic hand game usually played between two players. The rules are simple:

  1. Rock (Stone) beats Scissors.
  2. Scissors beats Paper.
  3. Paper beats Rock (Stone).

About project

Choices Available

The game has three possible moves: Stone (Rock), Paper, and Scissors.

Game Rules

  • Stone beats Scissors.
  • Scissors beats Paper.
  • Paper beats Stone
  • If both players choose the same, it’s a tie.

Game Logic

  • The program compares the user’s choice with the computer’s choice.
  • Based on the rules, it declares the result as Win, Lose, or Tie.

complete game Logic

  • Ask the user to enter their choice (Stone, Paper, or Scissors).

    • Computer’s Turn

  • The computer randomly selects its choice.

    • Comparison

  • If the user’s choice and the computer’s choice are the same → Tie
  • Otherwise, check the winning conditions:
    • Stone vs Scissors → Stone wins
    • Scissors vs Paper → Scissors wins
    • Paper vs Stone → Paper wins

    Code

    sample code of stone Paper Scissors game

    import random

def play_rps(user_choice):
    choices = ["Stone", "Paper", "Scissors"]
    computer_choice = random.choice(choices)
    print(f"Computer chose: {computer_choice}")

    if user_choice == computer_choice:
        return "It's a tie!"
    elif (user_choice == "Stone" and computer_choice == "Scissors") or \
         (user_choice == "Paper" and computer_choice == "Stone") or \
         (user_choice == "Scissors" and computer_choice == "Paper"):
        return "You win!"
    else:
        return "You lose!"

user_input = input("Enter your choice (Stone, Paper, Scissors): ")
result = play_rps(user_input)
print(result)
